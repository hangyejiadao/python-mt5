# python-mt5
python+mt5客户端实现策略回测

运行环境 Python3，mt5  
1、创建好Python3环境后再项目根目录执行 pip install -r requirements.txt 安装项目依赖的安装包  
2、demo.py 是回测需的入口文件，新策略请参考这个文件  


运行命令  
python demo.py
